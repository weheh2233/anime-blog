# Pomodoro Background Images

Place user-owned anime background images in this directory for later prototype tasks.

Task 1 intentionally renders a neutral placeholder when this directory has no image. Do not commit copyrighted images here unless you have the right to use them.

Suggested assets:
- Use your own or licensed bitmap images only.
- Prefer landscape images with the subject centered enough that the timer panel can stay readable over it.
- AVIF or WebP is preferred when you have it; PNG/JPG are fine for prototypes.
- If the directory stays empty, the app keeps showing the built-in neutral placeholder grid and rim.
